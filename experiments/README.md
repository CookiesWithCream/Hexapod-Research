# Experiments Folder

This folder contains selected experimental milestones only. Redundant bulk iterations were removed to keep the package manageable.

## Subfolders

- `ik_development_versions/` — selected IK mode-switch and Bézier development versions.
- `current_research_experimental/` — earlier research-style IK branch kept for reference.
- `legacy_control_iterations/` — selected key hardcoded/simultaneous/strafe control milestones.
- `legacy_server_control_iterations/` — selected web/server control experiments.
- `controller_client_iterations/` — selected controller client experiments.

These files are not the main runnable baseline. Use `control/` for current work.
