# Hardware Folder

This folder contains hardware-related project assets.

## Subfolders

- `cad_models/` — STL files for the robot body/leg parts. These are CAD/3D-printing reference files, not Python code.
- `legacy_robotis_motion_task_files/` — original ROBOTIS motion/task files kept for reference.

Temporary wiring reminders and personal future-task notes were removed from this cleaned package.
